## 已知问题

目前这货还不能算成品。基本是蛇每移动一次，就进行一次BFS找路。但是不能保证每一次都能找到一条路，找不到路的时候怎么办呢？我们让蛇随机走几步，走几步以后就有可能找到路了。但是这样做带来的后果就是：可能随机走着走着就走进死胡同了，结果只能GG。

改进的话：我想到一个点就是，BFS找不到路的时候，不随机走。而是有规律跟着尾巴走，比如做S型运动等。想法是这样，代码待写。
